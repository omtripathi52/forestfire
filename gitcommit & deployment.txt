Github:

1. Click on new repository and create repository

2. Open your project folder (the one you want to upload to GitHub)

3. Open the terminal in VS Code

   Shortcut: Ctrl + `

   Or: View â†’ Terminal

4. Check all files (including hidden ones)

   Run: ls -a

   If you see a folder named .git, Git is already initialized

   If not, initialize Git by running: git init

5. Check if any GitHub repository is already connected

   Run: git remote -v

   If nothing appears, continue

   If an old repository URL appears, remove it using: git remote rm origin

6. Connect your project to the new GitHub repository named forestfire

   Run: git remote add origin https://github.com/yourusername/forestfire.git

7. Verify the remote connection

   Run: git remote -v

   You should see the forestfire repository URL for fetch and push

8. Add all project files to the staging area

   Run: git add .

9. Commit the files

   Run: git commit -m "Initial commit: forestfire project"

10.   Push the project to GitHub

   Run: git branch -M main

   Run: git push -u origin main

Your project is now successfully uploaded to GitHub.
